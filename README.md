# product-planner-frontend

## Running with docker

1. `docker build -t product-planner-frontend .`
2. `docker run -it -p 8080:80 product-planner-frontend`

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
